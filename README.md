# tag-open-web
c++程序
这是一个帮你快速打开网页的东西。

此次版本号为1.0.0。

例子：

输入new 5 https://www.luogu.com.cn/

那么你就定义了一个叫5的tag，对应url是https://www.luogu.com.cn/ ，再次使用时，你可以输入5+一个字符串，这个字符串会被拼接在洛谷网址后面。
 
输入 5 team 
这就等于打开了一个url为https://www.luogu.com.cn/team 的网站。

还有一个命令为change，格式：change+tag（已定义）+url（更新）。
